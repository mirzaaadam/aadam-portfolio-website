
This repository contains the personal portfolio website I use for myself, modified from the Massively by HTML5 UP template, 
which can be found at html5up.net by @ajlkn (aj@lkn.io)
According to HTML5Up themselves these templates are free to use under the CCA 3.0 license (html5up.net/license)
The demo images, included by default with the theme, come from unsplash.com, while the icons re from fontawesome.io.

The three custom images on the website, were edited using pixlr editor from these two images (https://unsplash.com/photos/rBPOfVqROzY),
(https://unsplash.com/photos/5zp0jym2w9M), and (https://unsplash.com/photos/4qGbMEZb56c)
Other resources HTML5UP used in the creation of the website's templates are the following: 
		jQuery (jquery.com)
		Scrollex (github.com/ajlkn/jquery.scrollex)
		Responsive Tools (github.com/ajlkn/responsive-tools)
